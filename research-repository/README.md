# Orchid Research Repository

Repositorio documental y técnico para construir una base mundial de orquídeas, con énfasis en especies, sinónimos, híbridos y greges, genealogía, imágenes licenciadas, distribución, literatura histórica y conservación CITES.

## Principio central

Este repositorio no redistribuye libros ni bases protegidas por derechos de autor. Guarda enlaces, metadatos e identificadores estables y, únicamente cuando la licencia lo permite, archivos descargados.

## Fuentes prioritarias

1. RHS International Orchid Register: autoridad para híbridos y greges.
2. Kew POWO/WCVP e IPNI: nombres, taxonomía y nomenclatura.
3. GBIF: ocurrencias y conjuntos de biodiversidad.
4. iNaturalist: observaciones e imágenes según licencia individual.
5. Biodiversity Heritage Library: literatura histórica digitalizada.
6. CITES Species+ y Checklist: regulación y comercio internacional.

## Contenido inicial

- `data/sources.csv`: catálogo de fuentes.
- `data/library_seed.csv`: catálogo inicial de libros y publicaciones.
- `docs/LEGAL_AND_LICENSING.md`: reglas de uso.
- `docs/ROADMAP.md`: plan de ampliación.
